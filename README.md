Change Wallpaper <br>
    Usage:
    
    cw [arg1] [optional]
    
VID        = Only search and play videos        : Supported files gif, mp4, mkv, webm <br>  
IMG        = Only search and display images     : Supported files png, jpg, jpeg <br>
MUTE       = Optional mute video <br>

Keybinds: <br>
M          = Toggle mute <br>
K          = Kill mpvpaper <br>
Q/ESC      = Quit <br>
S          = Start swww daemon <br>
UP/DOWN    = Seek available <br>
ENTER      = Select as wallpaper <br>
