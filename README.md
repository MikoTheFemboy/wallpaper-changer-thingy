Change Wallpaper
    Usage:
    '''cw [arg1] [optional]'''
    VID        = Only search and play videos        : Supported files gif, mp4, mkv, webm
    IMG        = Only search and display images     : Supported files png, jpg, jpeg
    MUTE       = Optional mute video

    Keybinds:
    M          = Toggle mute
    K          = Kill mpvpaper
    Q/ESC      = Quit
    S          = Start swww daemon
    UP/DOWN    = Seek available
    ENTER      = Select as wallpaper

You're gonna need swww and mpvpaper (of course) and socat if you want to mute mpvpaper while it's playing :3
